## Instrucciones

1. Clona el repositorio.
2. Navega al directorio del proyecto.
3. Instala las dependencias:
   
 ```bash
   npm install
 ```

4. Ejecuta json-server:

   
 ```bash 
   npm run server
 ```


5. Ejecutar la aplicacion

   
 ```bash
   npm run dev
 ```
